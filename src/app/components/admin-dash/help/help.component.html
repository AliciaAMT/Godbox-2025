Posts Management

Creating and Managing Posts

Posts are the main content pieces on the platform. Each post can be configured with different privacy settings:

Privacy Settings:

  • Private: Only visible to the author. Won't show to other users.
  • Anonymous: Shows publicly but without author information.
  • Public: Shows everything - content and author information - to all users.

Post Sequencing:

To ensure posts appear in the correct order:

  1. Set the sequence number when creating or editing posts
  2. Lower numbers appear first (1, 2, 3, etc.)
  3. Posts with the same sequence number will be ordered by creation date

Collections

How Collections Work

Collections are groups of related posts that create a cohesive reading experience:

Collection Structure:

  • Cover Page: The post with sequence number 1 serves as the collection's cover page
  • Table of Contents: The cover page automatically includes a table of contents listing all posts in the collection
  • Menu Link: Collections create a menu link that users can access

Creating a Collection:

  1. Create multiple posts with the same collection name
  2. Set sequence numbers (1, 2, 3, etc.) for proper ordering
  3. The post with sequence 1 becomes the cover page
  4. All posts will be automatically linked together

Collection Display:

  • Posts appear in the table of contents in sequence order
  • Users can navigate through the collection using the cover page
  • Each collection creates its own menu item for easy access

Best Practices

Admin Dashboard Best Practices

Content Organization:

  • Use descriptive titles for posts and collections
  • Set appropriate privacy settings for each post
  • Use sequence numbers to control post order
  • Create collections for related content

User Experience:

  • Regularly review and moderate content
  • Monitor user engagement and feedback
  • Keep content fresh and relevant
  • Test navigation and user flows

Privacy Considerations:

  • Use private posts for draft content
  • Use anonymous posts for sensitive topics
  • Use public posts for general audience content
  • Review privacy settings before publishing
